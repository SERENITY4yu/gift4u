# Gift4U - Send Love Across Borders

A global gifting platform where anyone can send beautiful gifts (flowers, jewelry, pizza, frames, and more) to loved ones in the USA and Canada.

## Features

- Browse gifts by category (Flowers, Jewelry, Pizza, Frames, More)
- Add items to cart with quantity controls
- Checkout with recipient details and personal gift message
- Payment options: Stripe (Cards/Apple Pay/Google Pay), PayPal, Crypto
- Order confirmation with tracking info
- Fully responsive design
- Clean, colorful, and fun UI

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- React Context for state management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
